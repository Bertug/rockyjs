JSConf Espruino example

This is the example as seen at JSConf (YouTube).
It implements an animated watchface for Pebble entirely in JavaScript. Later during the presentation we used the very same JS source code to run it on a physical Pebble.

As our custom firmware at JSConf was based on Espruino and requires some specific APIs, this example also demonstrates how you can build different objects and methods on top of RockyJS to address your needs.
